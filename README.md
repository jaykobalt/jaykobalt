# My Page

GitHub Pages에 바로 올려 사용할 수 있는 정적 웹페이지입니다.

## 파일 구성

- `index.html`: 메인 페이지
- `styles.css`: 전체 스타일
- `assets/`: 썸네일 이미지 SVG

## GitHub Pages 배포 방법

1. 이 폴더를 GitHub 저장소에 업로드합니다.
2. 저장소에서 `Settings` → `Pages`로 이동합니다.
3. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
4. 브랜치를 `main`으로, 폴더를 `/ (root)`로 선택 후 저장합니다.
5. 잠시 후 배포 URL이 생성됩니다.

## 수정 포인트

- 소개 문구: `index.html`
- 메뉴 이름과 링크: `index.html`
- 색상과 여백: `styles.css`
- 썸네일 이미지 교체: `assets/` 내부 파일 교체 또는 `index.html` 경로 수정
